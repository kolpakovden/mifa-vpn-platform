# Code of Conduct

## Our Commitment

The MIFA project aims to maintain a welcoming and professional environment for all contributors and users.

We expect all participants in the community to engage respectfully and constructively.

## Expected Behavior

Participants are expected to:

- Use welcoming and inclusive language
- Be respectful of differing viewpoints and experiences
- Accept constructive criticism
- Focus on what is best for the project and the community

## Unacceptable Behavior

The following behaviors are not tolerated:

- Harassment or discriminatory language
- Personal attacks
- Publishing private information without permission
- Any behavior that would be considered inappropriate in a professional setting

## Enforcement

Project maintainers have the right to remove, edit, or reject comments, commits, issues, and contributions that violate this Code of Conduct.

## Scope

This Code of Conduct applies to all project spaces, including:

- GitHub issues
- pull requests
- discussions
- project communication channels

## Reporting

If you experience or witness unacceptable behavior, please report it through the repository issue tracker or by contacting the maintainers.
